# Reverse A Queue

---

## Aproach 1 (Stack Approach)

ðŸ‘‰ Time -> O(n)  & Space -> O(n)

```cpp
void reverseQueue(queue<int> &q){
    stack<int> s;
    
    // push each element from queue to stack
    while(!q.empty()){
        int el = q.front();
        q.pop();
        s.push(el);
    }
    
    // push each element from stack to queue
    while(!s.empty()){
        q.push(s.top());
        s.pop();
    }

}
```

## Aproach 2 ( Recursion )

ðŸ‘‰ Time -> O(n)  & Space -> O(1)

```cpp
void reverseQueue(queue<int> &q){
    // base case
    if(q.empty()){
        return;
    }
    
    int el = q.front();
    q.pop();
    
    // recursion call
    reverseQueue(q);
    
    q.push(el);
}
```
